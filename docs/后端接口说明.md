# 前后端接口说明

## 数据表

- `users`：登录账号、密码哈希和角色。
- `devices`：ESP32-S3 设备、所属用户、设备密钥及在线状态。
- `detection_events`：设备上报的姿态检测事件。
- `daily_stats`：按用户和日期聚合的分钟数、提醒次数及健康评分。

## 核心接口

| 方法 | 路径 | 鉴权 | 用途 |
| --- | --- | --- | --- |
| POST | `/api/auth/login` | 公开 | 登录并获取 JWT |
| GET | `/api/stats/today` | Bearer JWT | 今日汇总 |
| GET | `/api/events/recent` | Bearer JWT | 最近检测事件 |
| POST | `/api/device/events` | `X-Device-Key` | ESP32-S3 上报事件 |
| GET | `/api/stats/posture-trend` | Bearer JWT | 最近若干天趋势 |

此外提供注册、当前用户、改密和管理员查询接口，均可在 `/docs` 查看。

## 设备上报示例

```json
{
  "event_type": "head_down",
  "is_present": true,
  "risk_level": 2,
  "head_angle": -18.5,
  "duration_minutes": 5
}
```

合法事件类型：`normal`、`away`、`head_down`、`lean_forward`、`head_tilt`、`long_sitting`、`phone_detected`。
