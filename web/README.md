# Web Dashboard Module

This folder contains the frontend dashboard for DeskEye Guard.

## Responsibilities

- Display current desktop behavior state
- Show sitting-time timer
- Show warning history
- Show daily summary statistics
- Provide simple settings interface

## Possible Tech Stack

Early prototype:

- HTML + CSS + JavaScript

Later version:

- React or Vue
- Chart library for statistics
- WebSocket for real-time state updates

## First Milestone

The first web milestone is to build a simple page that can display current state values such as `normal`, `away`, `too_close`, and `long_sitting`.
